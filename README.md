# Servo Controller
Dedicated servo controller developed to offload real time actuator control from the main compute system while maintaining robust communication over CAN.
<br>

- CAN bus communication for control and fault reporting
- Multi channel PWM servo outputs with precise timing

## Board Layout
PCB layout focused on signal integrity, and power stability.
<br>

- Isolated power routing for servos and logic
- Short, matched signal paths for consistent PWM timing
- Clear connector placement for harnessing and serviceability
